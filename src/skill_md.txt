# localflow

ローカルタスク管理CLI。プロジェクトのタスクをSQLiteデータベースで管理する。

## グローバルオプション

| オプション | 説明 |
|---|---|
| `--output <FORMAT>` | 出力形式。`json`（デフォルト）または `text` |
| `--project-root <PATH>` | プロジェクトルートディレクトリ。省略時はカレントディレクトリ |

## コマンド

### `localflow add`

新しいタスクを追加する。

### `localflow list`

タスク一覧を表示する。

### `localflow get`

タスクの詳細を表示する。

### `localflow next`

次に着手すべきタスクを表示する。優先度・依存関係を考慮して自動選択する。

### `localflow edit`

既存タスクを編集する。

### `localflow complete`

タスクを完了にする。

### `localflow cancel`

タスクをキャンセルする。

### `localflow deps add <TASK_ID> --on <DEP_ID>`

タスクに依存関係を追加する。循環依存・自己依存はエラーになる。

### `localflow deps remove <TASK_ID> --on <DEP_ID>`

タスクの依存関係を削除する。

### `localflow deps set <TASK_ID> --on <DEP_ID>...`

タスクの依存関係を全置換する。既存の依存をすべて削除し、指定したIDで再設定する。

### `localflow deps list <TASK_ID>`

タスクの依存先タスク一覧を表示する。

### `localflow skill-install`

Claude Code 用の SKILL.md ファイルを生成する。

| オプション | 説明 |
|---|---|
| `--output-dir <PATH>` | 出力先ディレクトリ。省略時はカレントディレクトリ |

## ステータス

| 値 | 説明 |
|---|---|
| `pending` | 未着手 |
| `in_progress` | 作業中 |
| `done` | 完了 |
| `cancelled` | キャンセル済み |

## 優先度

| 値 | 説明 |
|---|---|
| `low` | 低 |
| `medium` | 中（デフォルト） |
| `high` | 高 |
| `critical` | 最優先 |

## 依存関係

タスクは他のタスクへの依存関係を持つことができる。依存先タスクが完了するまで、依存元タスクは `next` コマンドで選出されない。
